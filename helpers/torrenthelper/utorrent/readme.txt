Помогает скачать торрент в указанную папку.
Вызов должен происходить неповсредственно из модуля, связанного с определенным трекером.
Скрипт download.py принимает в качестве аргументов:

	1. Путь к .torrent файлу
	2. Путь для сохранения файлов
	
Для вызова скрипта OnEndLoad.py внутри папки с медиа 
необходимо добавить (Путь к OnEndLoad.bat) %D 
в UTorrent->Настройки->Настройки программы->Дополнительно->Запуск программы->Запустить программу по завершении торрента.
